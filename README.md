# ai-notes

This repository stores AI lecture notes organized by locale.

## Structure

- `matt-pocock-ai-workflow/`
  - `matt-pocock-ai-workflow-ru.md` — Russian version of the Matt Pocock AI workflow notes
  - `matt-pocock-ai-workflow-en.md` — English version of the Matt Pocock AI workflow notes

## How to use

- Open the folder for a lecture.
- Select the locale-specific markdown file with the language suffix.
- Add new lectures as new folders, each containing locale-specific files.

## Add a new lecture

1. Create a new folder named after the lecture.
2. Add files with locale suffixes, for example:
   - `lecture-name-ru.md`
   - `lecture-name-en.md`
3. Update this README if you want to list the new lecture.
